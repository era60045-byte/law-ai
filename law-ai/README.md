\# Law AI



Проект для первичного анализа нормативных текстов.



\## Что делает

Система сравнивает тексты норм и определяет:

\- возможные дубли

\- похожие нормы

\- возможные противоречия



\## Как запустить

1\. Установить зависимости:

pip install -r requirements.txt



2\. Запустить:

python main.py



\## Используемые данные

В текущем MVP используются тестовые тексты нормативных норм.



\## Ограничения

Текущая версия работает на небольшом количестве текстов и использует базовый подход анализа.
